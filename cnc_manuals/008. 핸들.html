핸들 사용 방법 (HANDLE 사용 방법)

페이지 58

  1. 자막 해설:

    • "HANDLE 사용법 시나리오 시작"
    • 조작판 앞 위치에서 Scene 시작

  2. 나레이션:

    • "HANDLE 사용법에 대해 알아보겠습니다."

  3. 음성 해설:

    • "핸들 사용법에 대해 알아보겠습니다."

  4. 나레이션:

    • "HANDLE MODE를 선택해주세요."

  5. 음성:

    • "먼저 핸들 모드를 선택해주세요."

  6. 버튼 표시 및 빛깔:

    • HANDLE 버튼이 노란 빛으로 깜빡인다.

  7. 상세 설명:

    • HANDLE MODE를 선택하면 나레이션 시작

페이지 59

  1. 나레이션:

    • "HANDLE AXIS 항목 중 X 또는 Z 버튼을 눌러서 이동할 축을 선택해주세요."

  2. 음성 해설:

    • "축을 선택하시려면 중에서 하나의 축을 선택해주세요."

  3. 버튼 표시 및 빛깔:

    • X 또는 Z 버튼이 빛으로 깜빡인다.

  4. 상세 설명:

    • X/Z 축 선택 시 나레이션 이어짐

페이지 60

  1. 나레이션:

    • "X1, X10, X100 중에 하나를 선택하여 이동량을 선택해주세요."

  2. 음성 해설:

    • "X1, X10, X100 중에 하나를 선택하여 이동량을 설정해주세요."

  3. 버튼 표시 및 빛깔:

    • X1, X10, X100 버튼 중 하나가 빛으로 깜빡인다.

  4. 상세 설명:

    • 이동량 선택 시 나레이션 이어짐

페이지 61

  1. 나레이션:

    • "조작판 왼쪽에 보이는 HANDLE에 외관상 생긴 그림자를 가까이 가져가서 다음 그림 버튼을 눌러 HANDLE을 잡아주세요."

  2. 음성 해설:

    • "조작판 왼쪽에 보이는 핸들에 외관상이 생긴 그림자를 가까이 가져가서 다음 그림 버튼을 눌러주세요."

  3. 버튼 표시 및 빛깔:

    • HANDLE의 그림자가 생기며 빛으로 깜빡인다.

  4. 상세 설명:

    • HANDLE을 잡으면 다음 장면 나레이션 시작

페이지 62

  1. 나레이션:

    • "이제 HANDLE을 잡은 상태로 안전거리값으로 스택 좌, 우로 인해서 HANDLE을 돌려보세요."

  2. 음성 해설:

    • "이제 핸들을 잡은 상태로 안전거리값으로 스택 좌, 우로 돌려보세요."

  3. 상세 설명:

    • 스택 좌우로 핸들을 돌리다 멈추면 시나리오 종료
