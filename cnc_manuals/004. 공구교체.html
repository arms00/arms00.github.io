올린 페이지들(22-32페이지)의 내용을 공구 교체라는 주제 아래 최대한 상세하게 분석하고 정리한 내용을 아래에 제공합니다:

공구 교체 단계- MDI MODE (페이지 22-32)

1. 시작 단계

  • 자막해설: "공구 교체 시나리오 시작"
  • 내레이션: "CNC 선반에서 공구 교체하는 방법을 알아보겠습니다."
  • 음성: "CNC 선반에서 공구 교체하는 방법을 알아보겠습니다."
  • 내레이션: "MDI MODE을 선택해주세요."
  • 음성: "MDI MODE을 선택해주세요."
  • 버튼 생성: 각종 모드 선택을 위한 MDI 버튼 생성

2. 프로그램(PROG) 버튼 사용

  • 내레이션: "PROG 버튼을 눌러주세요."
  • 음성: "프로그램 버튼을 눌러주세요."
  • 버튼 표시: PROG 버튼이 생성되며, 이를 눌러 다음 단계로 이동

3. MDI 모드 활성화

  • 내레이션: "컨트롤러 화면 하단에 MDI 버튼을 누르면 코드 입력창이 나타납니다."
  • 음성: "컨트롤러 화면 하단에 엠디아이 버튼을 누르면 코드 입력창이 나타납니다."
  • MDI 버튼: 해당 버튼을 눌러 코드를 입력할 수 있는 창이 표시됨
  • MDI 버튼 클릭 후: 다음 단계로 이동

4. 코드 입력 (T0300)

  • 내레이션: "설정된 보정값을 적용하지 않은 상태에서 3번 공구로 교체를 하기 위해 조작반에서 T0300;를 입력해주세요."
  • 음성: "설정된 보정값을 적용하지 않은 상태에서 상단 T0300을 입력해주세요."
  • T0300 입력: 3번 공구를 교체하기 위한 코드 T0300을 입력하고 나서 다음 단계로 이동

5. 코드 실행 (INSERT 버튼)

  • 내레이션: "INSERT 버튼을 눌러 입력한 코드를 위로 올려줍니다."
  • 음성: "위로 올린 후 INSERT 버튼을 누릅니다."
  • INSERT 버튼: 입력한 코드를 실행하기 위해 해당 버튼을 눌러 완료

6. 코드 재입력 시 오류

  • 내레이션: "T0300;가 아닌 다른 코드를 기입한 후 INSERT 버튼을 누를 때 입력된 코드는 삭제되면서 다음 자막, 나레이션 시작."
  • 내레이션: "T0300; 를 입력하여 주세요."
  • 음성: "T0300; 를 입력하여 주세요."
  • 잘못된 코드: T0300 대신 다른 코드가 입력되면 다시 올바른 코드를 입력하도록 유도

7. 사이클 시작

  • 내레이션: "이제 CYCLE START 버튼을 누르면 공구가 재설치됩니다."
  • 음성: "이제 싸이클 스타트 버튼을 누르면 공구가 교체됩니다."
  • CYCLE START 버튼: 공구가 교체된 후 사용을 시작하는 버튼으로 활성화됨
  • 내레이션: CYCLE START 버튼을 눌러 공구 교체가 완료됨

분석된 주요 항목:

  1. MDI 모드 사용: CNC 선반에서 공구를 교체하기 위해 MDI 모드를 사용하며, 사용자는 PROG 버튼을 눌러 코드를 입력할 수 있습니다.
  2. 코드 입력: 공구 교체에 필요한 명령어 (T0300, T0101 등)를 입력하여 공구를 교체하는 과정을 거침.
  3. 버튼 상호작용: MDI 모드에서 주요한 버튼(INSERT, PROG, CYCLE START 등)을 눌러 사용자가 교체 과정을 진행.
  4. CYCLE START: 교체된 공구를 사용하기 위한 마지막 단계로, CYCLE START 버튼을 통해 교체된 공구가 실제로 적용됨.
  5. 내레이션과 자막: 사용자가 혼동하지 않도록 모든 단계에서 적절한 내레이션과 자막을 통해 명확한 설명 제공.

이와 같은 공구 교체 과정은 일관되게 설명되며, 사용자가 CNC 선반에서 공구 교체 작업을 쉽게 이해하고 실행할 수 있도록 돕는 구조로 되어 있습니다.
