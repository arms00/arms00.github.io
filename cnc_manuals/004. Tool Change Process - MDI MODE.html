Tool Change Process - MDI MODE (Pages 22-32)

The process outlines how to change a tool in a CNC lathe using the MDI (Manual Data Input) mode. Users will go through each step, activating MDI mode, inputting the necessary tool codes, and applying the changes.

1. Initial Step

  • Subtitles: "Tool Change Scenario Begins"
  • Narration: "We will now learn how to change tools on a CNC lathe."
  • Voice-over: "We will now learn how to change tools on a CNC lathe."
  • Narration: "Please select MDI MODE."
  • Voice-over: "Please select MDI MODE."
  • Button Generation: MDI mode button is created for the user to activate the mode.

2. Program (PROG) Button Use

  • Narration: "Please press the PROG button."
  • Voice-over: "Please press the program button."
  • Button Highlight: The PROG button lights up, and the user presses it to move to the next step.

3. Activate MDI Mode

  • Narration: "Press the MDI button on the controller's lower screen, and a code input window will appear."
  • Voice-over: "Press the MDI button on the controllerâ€™s screen, and a code input window will appear."
  • MDI Button: After pressing this button, a code input window is displayed.
  • MDI Button Interaction: Pressing the MDI button moves to the next step.

4. Code Input (T0300)

  • Narration: "To change to tool 3 without applying any preset compensation values, please input T0300; at the control panel."
  • Voice-over: "Please input T0300 at the control panel to change to tool 3 without preset compensation values."
  • Code Entry: Input T0300 to command a tool change to tool 3 and proceed to the next step.

5. Execute Code (INSERT Button)

  • Narration: "Press the INSERT button to move the input code upward."
  • Voice-over: "Press the INSERT button to confirm the input code."
  • INSERT Button: After inputting the code, pressing the INSERT button confirms and applies the command.

6. Error When Inputting Incorrect Code

  • Narration: "If you enter a code other than T0300 and press the INSERT button, the entered code will be erased, and a new narration and subtitle will prompt you."
  • Narration: "Please input T0300;."
  • Voice-over: "Please input T0300."
  • Error Correction: If an incorrect code is entered, the system prompts the user to correct it by inputting T0300 again.

7. Cycle Start (Final Confirmation)

  • Narration: "Now press the CYCLE START button to apply the tool change."
  • Voice-over: "Press the cycle start button to apply the tool change."
  • Cycle Start Button: The tool change is finalized by pressing the cycle start button, and the tool is now ready for use.
  • Completion: The tool change process finishes after the button is pressed and the new tool is successfully applied.

Analysis of Key Points:

  1. Use of MDI Mode: The tool change process on the CNC lathe requires MDI (Manual Data Input) mode, where the user interacts with the program and manually inputs the necessary tool codes.
  2. Code Input: Essential tool-changing commands like T0300 and T0101 are manually entered by the user to switch tools in the machine.
  3. Interaction with Buttons: Several key buttons (PROG, INSERT, CYCLE START) guide the user through the process, ensuring that the tool change is both commanded and executed properly.
  4. Cycle Start: The process concludes with the CYCLE START button, which applies the tool change and allows the new tool to be used in subsequent operations.
  5. Error Handling: The user is guided through error correction when incorrect codes are entered, ensuring a smooth and correct tool change process.
  6. Narration and Subtitles: Both narration and subtitles provide clear instructions for each step to avoid confusion, enhancing the learning experience.
