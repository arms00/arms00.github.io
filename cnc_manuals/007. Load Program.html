화면 설명 및 시나리오

  1. 장면 ID: G_EDIT MODE (52페이지)

    • 화면 구성 요소 설명:
      1. 3D: 화면에 표시된 3D 구성 요소 (선반 컨트롤러 앞).
      2. 나레이션: "프로그램 호출하는 방법에 대해 알아보겠습니다."
      3. 음성: 나레이션과 동일한 내용으로 출력.
      4. 팝업창: 필요한 시각적 피드백이 제공됨.
      5. Btn: 필요한 버튼들에 대한 안내 제공 (EDIT 버튼).
      6. 자막해설: 프로그램 호출 시나리오 진행에 따른 추가 설명 제공.
    • 세부 내용:
      1. 자막해설: "프로그램 호출"
      2. 나레이션: "프로그램 호출하는 방법에 대해 알아보겠습니다."
      3. 음성: 나레이션과 동일한 내용.
      4. 나레이션: "우선 EDIT MODE를 선택해주세요."
      5. Btn: EDIT MODE 버튼을 누르면 다음 장면에서 시나리오 진행.

  2. 장면 ID: G_EDIT MODE (53페이지)

    • 화면 구성 요소 설명:

      1. 3D: CNC 선반 컨트롤러 화면 표시.
      2. 나레이션: "PROG 버튼을 눌러주세요."
      3. 음성: 동일한 내용 출력.
      4. Btn: PROG 버튼에 대한 강조.
      5. 자막해설: 시나리오 진행에 따른 안내.

    • 세부 내용:

      1. 나레이션: "PROG 버튼을 눌러주세요."
      2. 음성: 동일한 내용.
      3. Btn: PROG 버튼 선택 후 다음 장면에서 시나리오 진행.

  3. 장면 ID: G_EDIT - DIR (54페이지)

    • 화면 구성 요소 설명:

      1. 3D: 컨트롤러 화면 하단의 DIR 버튼 선택.
      2. 나레이션: "DIR 버튼을 눌러 프로그램 목록을 확인할 수 있습니다."
      3. 음성: 동일한 내용 출력.
      4. Btn: DIR 버튼 강조 및 선택.
      5. 자막해설: DIR 버튼 선택 후 다음 과정 안내.

    • 세부 내용:

      1. 나레이션: "DIR 버튼을 눌러 프로그램 목록을 확인할 수 있습니다."
      2. 음성: 동일한 내용.
      3. Btn: DIR 버튼 선택 후 다음 장면에서 나레이션 시작.

  4. 장면 ID: G_EDIT - DIR (55페이지)

    • 화면 구성 요소 설명:

      1. 3D: 프로그램 호출을 위한 코드 입력 과정.
      2. 나레이션: "프로그램 Q2016을 호출하기 위해 00000을 입력해주세요."
      3. 음성: 동일한 내용.
      4. 팝업창: 입력한 코드가 반영된 화면.
      5. 자막해설: 코드를 입력한 후 다음 과정으로 안내.

    • 세부 내용:

      1. 나레이션: "프로그램 Q2016을 호출하기 위해 00000을 입력해주세요."
      2. 음성: 동일한 내용.
      3. 코드 입력 후 다음 장면에서 시나리오 진행.

  5. 장면 ID: G_EDIT - DIR (56페이지)

    • 화면 구성 요소 설명:

      1. 3D: 검색 버튼 선택 과정.
      2. 나레이션: "검색 버튼을 눌러 선택한 프로그램을 호출합니다."
      3. 음성: 동일한 내용.
      4. Btn: 검색 버튼에 대한 강조.

    • 세부 내용:

      1. 나레이션: "검색 버튼을 눌러 선택한 프로그램을 호출합니다."
      2. 음성: 동일한 내용.
      3. Btn: 검색 버튼 선택 후 다음 장면에서 시나리오 진행.

  6. 장면 ID: G_EDIT - DIR (57페이지)

    • 화면 구성 요소 설명:

      1. 3D: CNC 컨트롤러 화면에 선택한 프로그램 실행 완료.
      2. 나레이션: "선택한 프로그램이 호출되었습니다."
      3. 음성: 동일한 내용.
      4. 자막해설: 프로그램 호출 시나리오 종료.

    • 세부 내용:

      1. 나레이션: "선택한 프로그램이 호출되었습니다."
      2. 음성: 동일한 내용.
      3. 시나리오 종료.

위의 정리는 각 페이지에 나와 있는 모든 텍스트와 시각적 요소를 분석하고 정리한 것입니다. "프로그램 호출 방법"이라는 주제 아래 각 단계별로 화면 구성과 나레이션에 맞춘 설명이 빠짐없이 정리되었습니다.
